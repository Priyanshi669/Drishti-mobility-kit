# Drishti Mobility Kit - Smart Stick Components

## 1. Core Electronics
- ESP32 Microcontroller
- Ultrasonic Sensor (HC-SR04)
- IR Obstacle Sensor (optional)
- GPS Module (NEO-6M or equivalent)
- Bluetooth Module (if not using ESP32 internal BT)
- Buzzer
- Vibration Motor
- Li-ion Battery (18650) or 9V rechargeable
- Step-down regulator (5V)

## 2. Frame & Outer Body
- PVC tube (lightweight)
- 3D printed mount for electronics (optional)
- Holder for vibration motor

## 3. Wires & Accessories
- Jumper wires
- Heat shrink tube
- Insulation tape
- Power switch

## 4. Tools Used
- Soldering iron
- Glue gun
- Wire stripper
